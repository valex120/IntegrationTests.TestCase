Get
1. Get. Передан идентификатор несуществующего товара. Возвращает 404 NotFound "ProductNotFound".
2. Get. Передан идентификатор существующего товара. Возвращает товар.

List
1. List. Передана неизвестная категория товара. Возвращает 400 BadRequest, "UnknownCategory".
2. List. Передана начальная цена 0. Возвращает 400 BadRequest, "ZeroStartPrice".
3. List. Передана начальная цена -1. Возвращает 400 BadRequest, "NegativeStartPrice".
4. List. Передана конечная цена 0. Возвращает 400 BadRequest, "ZeroEndPrice".
5. List. Передана конечная цена -1. Возвращает 400 BadRequest, "NegativeEndPrice".
6. List. Передана начальная цена 10, конечная цена 5. Возвращает 400 BadRequest, "StartPriceMoreThenEndPrice".
7. List. Передана начальная цена 10. В БД есть товар с ценой 11. Возвращает товары с ценой более 10, включая заведённый.
8. List. Передана конечная цена 10. В БД есть товар с ценой 9. Возвращает товары менее 10, включая заведённый.
9. List. Передана категория товара Children. В БД есть товар категории Children. Возвращает товары категории Children, включая заведённый.
10. List. Передана вхождение в название "Книга". В БД есть товар с "Книга" в названии. Возвращает товары с "Книга" в названии, включая заведённый.

Update
1. Update. Передано пустое имя товара. Возвращает 400 BadRequest, "EmptyName".
2. Update. Передано имя товара длиной 1001 символ. Возвращает 400 BadRequest, "NameTooLong".
3. Update. Передана нулевая цена. Возвращает 400 BadRequest, "ZeroPrice".
4. Update. Передана отрицательная цена. Возвращает 400 BadRequest, "NegativePrice".
5. Update. Передана неизвестная категория товара. Возвращает 400 BadRequest, "UnknownCategory".
6. Update. Передан идентификатор несуществующего товара. Возвращает 404 NotFound "ProductNotFound".
7. Update. Переданы корректные данные. Обновляет данные товара.

Delete
1. Delete. Передан идентификатор несуществующего товара. Возвращает 404 NotFound "ProductNotFound".
2. Delete. Передан идентификатор существующего товара. Удаляет товар.
